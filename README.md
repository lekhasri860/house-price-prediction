# House Price Prediction Project

This is my end-to-end Machine Learning project to predict house prices in Vijayawada using features like area, bedrooms, bathrooms, age, location, and property type.

## How to Run

1. Install dependencies:
 pip install -r requirements.txttext
2. Run the app:
   python src/web_app.pytext.
3.  Open in browser:
http://127.0.0.1:5000

## Features
- Web form for input  
- XGBoost model for prediction  
- Input validation & error messages  
- Simple UI with results in green box

## Files Overview
- `src/web_app.py` → Flask web app  
- `templates/index.html` → form page  
- `models/best_model.joblib` → trained model  
- `data/house_prices.csv` → dataset  
- `requirements.txt` → dependencies
